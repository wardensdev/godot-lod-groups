# godot-lod-groups
An example implementation of LOD Groups in Godot 4.0 Alpha.

# Using Test Project
- Download/clone the files into a new Godot project.
- Set Node3D.tscn as the main scene.
- Run the project.
- Up/Down arrow keys to move the camera forward and backward.

# In the works
